# ようこそ、あなたのナレッジベースへ
<!-- updated: 2026-04-12 -->

このファイルが見えているなら、Obsidian で Vault を開くことに **成功** しています。

## まず試してほしいこと

### 1. 目次を開く
[[index|ナレッジベース インデックス]] を開いて全体像を確認してください。

### 2. 概念ページを見る
主要な10個の概念ページが既に用意されています。

- [[部活動の地域展開]]
- [[教育委員会運営]]
- [[施設管理]]
- [[予算管理]]
- [[人権研修]]
- [[家庭菜園]]
- [[家族の日常]]
- [[自己分析と成長]]
- [[AI活用]]
- [[投資と経済]]

### 3. グラフビューを開く
左サイドバーの **グラフアイコン** （3つの点が繋がった図）をクリックすると、全ページの関連図が可視化されます。

グラフの色分け（設定済み）:
- **青系**: 概念ページ
- **赤系**: エンティティ（人物・組織）
- **紫系**: ソース要約
- **オレンジ系**: 分析結果

### 4. 分析結果を確認
- [[2026-04-12-明日やるべきタスクtop10|明日やるべきタスク TOP 10]]

### 5. 検索を使う
`Ctrl + Shift + F`（Mac: `Cmd + Shift + F`）で全文検索が開きます。例えば「部活動」「いなチャレ」「家庭菜園」などで試してください。

## Obsidian の基本操作

| ショートカット | 動作 |
|---------------|------|
| `Ctrl + O` | ファイルを素早く開く |
| `Ctrl + P` | コマンドパレット |
| `Ctrl + Shift + F` | 全文検索 |
| `Ctrl + E` | 編集⇔プレビュー切替 |
| `Ctrl + Click` | リンク先を新タブで開く |
| `Alt + ←/→` | 履歴を戻る/進む |

## リンクの書き方

```
[[ページ名]]              ← そのページへのリンク
[[ページ名|表示テキスト]] ← 表示テキストを変える
[[ページ名#見出し]]       ← 特定の見出しへのリンク
```
