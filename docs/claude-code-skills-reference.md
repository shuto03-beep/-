# Claude Code 活用スキル・リファレンス

## 1. Superpowers（開発プロセス強化）
プロエンジニアのワークフロー（TDD、設計、計画、実行、レビュー）を強制するプラグイン。
- **特徴**: 要件定義の「壁打ち」が非常に丁寧。曖昧な指示をプロレベルの設計書に落とし込める。
- **主要コマンド**:
  - `/superpowers/brainstorm`: アイデアを設計に落とし込む
  - `/superpowers/plan`: 実装計画の作成
  - `/superpowers/execute`: 計画に基づいた実行
- **導入メリット**: 開発経験が浅いユーザーでも、正しい順序で堅牢なコードを生成できる。

## 2. Impeccable（デザイン品質向上）
AI特有の「派手すぎるグラデーション」や「不自然な配置」を排除し、モダンなUIを作る。
- **特徴**: デザイン原則（トークン、システム）に基づいた評価と修正が可能。
- **主要コマンド**:
  - `/te-impeccable`: デザインの方向性を教え込む（セットアップ）
  - `/critic`: 現状のUIをUX/UIの観点から数値で評価し、改善案を出す
  - `/polish`: 細かなデザインの微調整を行う
- **活用法**: `/critic`で自己評価（例：40点中26点）をさせてから、その課題をすべて一気に修正させる。

## 3. Everything Claude Code (ECC)（環境最適化）
Claude Codeを10ヶ月使い倒したエキスパートの環境セット（エージェント、スキル、フック）の集合体。
- **特徴**: フォルダ構成の自動化やセキュリティスキャン、継続的な学習機能を含む。
- **注意点**: 機能が多いため、Claude Codeの基本操作に慣れてから導入するのが推奨。

## 4. Find Skills（スキル検索・管理）
Vercel Labsが提供する、スキルのエコシステム管理ツール。
- **特徴**: CLI（ターミナル）上から新しいスキルを探してインストールできる。
- **URL**: [skills.sh](https://skills.sh)

「以下のスキルをインストールし、[いなチャレ事務局]の開発を進めてください。
